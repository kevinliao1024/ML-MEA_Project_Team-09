## 1. ESOL（溶解度预测）—— 🟢 优秀

**任务描述**：预测分子在水中的溶解度（log mol/L），是药物设计中的关键属性。

### 评估指标详解

| 指标 | 数值 | 中文名 | 统计意义 |
|------|------|--------|---------|
| **Pearson r** | 0.9487 | 皮尔逊相关系数 | 衡量预测值与真实值的线性相关程度，接近1表示强正相关。r > 0.9 说明模型能很好地捕捉溶解度与分子结构之间的线性关系。 |
| **Spearman r** | 0.9470 | 斯皮尔曼秩相关系数 | 衡量单调相关性，对异常值不敏感。r > 0.9 表明预测排序与真实排序高度一致。 |
| **R²** | 0.8992 | 决定系数 | 模型能解释89.9%的数据方差。R² > 0.85 即为优秀，本结果达到高水平。 |
| **MSE** | 0.4309 | 均方误差 | 误差平方的均值，对大误差惩罚更重。回归任务专用指标，越小越好。 |
| **MAE** | 0.4692 | 平均绝对误差 | 平均每个预测的绝对误差幅度。本任务溶解度数值约在-13到1之间，0.47的误差属于高精度。 |

### 结论

> ESOL 是回归任务中表现最稳定的指标组。三个相关性指标均 > 0.94，R² = 0.90，表明 Uni-Mol 对分子溶解度预测具有很强的能力。MSE/MAE 在合理范围内，平均误差约 0.47 个 log 单位。

## 2. BBBP（血脑屏障穿透性预测）—— 🟢 良好

**任务描述**：预测分子能否穿透血脑屏障（BBB），是中枢神经系统药物设计的核心筛选指标。

### 评估指标详解

| 指标 | 数值 | 中文名 | 统计意义 |
|------|------|--------|---------|
| **AUC / AUROC** | 0.8985 | 曲线下面积 | 衡量模型区分正负样本的整体能力。1.0=完美分类，0.5=随机猜测，> 0.8=良好，> 0.9=优秀。本模型达到良好水平。 |
| **AUPRC** | 0.9737 | 精确率-召回率曲线下面积 | 在不平衡数据集下衡量预测质量，比AUC更敏感。本指标极高，说明在各类别上均有稳定表现。 |
| **Accuracy** | 0.9056 | 准确率 | 正确预测的比例（正类+负类均正确）。90.6% 的预测是正确的。 |
| **F1-Score** | 0.9452 | F1 分数 | 精确率和召回率的调和平均，兼顾两者。> 0.9 为优秀水平。 |
| **Precision** | 0.9216 | 精确率 | 预测为正类（可穿透BBB）的样本中，真正为正的比例。92.2% 的预测"可穿透"是正确的。 |
| **Recall** | 0.9701 | 召回率 | 真正为正的样本中，被正确预测出来的比例。97% 的真正可穿透分子被成功识别。 |
| **MCC** | 0.6166 | 马修斯相关系数 | 考虑全部四类预测组合的相关系数，取值 -1 到 1，0 为随机。> 0.6 为中等相关，说明模型在数据不平衡时的一致性仍有提升空间。 |
| **Cohen's Kappa** | 0.6061 | 科恩卡帕系数 | 去除随机因素后的实际一致率，> 0.6 为中等一致。说明模型预测与真实标签的一致性达到中等水平。 |
| **Log Loss** | 0.3676 | 对数损失 | 预测概率分布与真实标签的交叉熵，越小越好。本指标衡量概率预测的质量。 |

### 结论

> BBBP 的 AUC = 0.90 表明模型具有良好的分类能力。精确率和召回率都很高（均 > 92%），F1 = 0.9452 达到优秀水平。但 MCC 和 Kappa 处于中等水平（~0.61），说明在数据不平衡场景下仍有优化空间。

## 3. Tox21（21种毒性多标签预测）—— 🟡 偏弱

**任务描述**：预测分子在21种不同毒性测试中的阳性/阴性结果，包括 Nuclear receptor signaling 和 Stress response pathways 两大类。

### 评估指标详解

| 指标 | 数值 | 中文名 | 统计意义 |
|------|------|--------|---------|
| **AUC / AUROC** | 0.6475 | 曲线下面积 | 区分毒性/非毒性的能力，0.65 仅略高于随机（0.5），表明模型对多数毒性标签的区分能力较弱。 |
| **AUPRC** | 0.2377 | 精确率-召回率曲线下面积 | 在极端不平衡的多标签数据下（正样本极少），AUPRC 比 AUC 更能反映真实性能。0.24 处于很低水平。 |
| **Accuracy** | 0.9624 | 准确率 | **表面优秀，实为假象**。21个标签中绝大多数为阴性（无毒），模型只需预测"无毒"即可达到 96% 以上准确率。 |
| **MCC** | 0.3306 | 马修斯相关系数 | 多标签场景下的综合相关性指标，> 0.3 为弱相关，0.33 处于弱相关区间的上沿。 |
| **Log Loss** | 0.4888 | 对数损失 | 概率预测质量，在 0~1 之间越小越好。本指标偏高，说明概率校准有待改进。 |

### ⚠️ 关键洞察：准确率的陷阱

> Tox21 的 **准确率 96.2% 是假象**。21个标签中约 90% 为阴性，模型只需始终预测"无毒"即可达到 ~90% 的准确率。真正反映模型能力的指标是 **AUC = 0.65**（接近随机上限 0.5）和 **MCC = 0.33**（弱相关），两者均处于偏弱水平。

### 可能原因

1. **数据不平衡**：Tox21 中部分标签正样本极少（如某些 nuclear receptor 标签正样本 < 5%）
2. **标签数量多**：21个标签增加了学习难度
3. **毒性机制复杂**：部分毒性依赖生物代谢通路，2D分子结构信息不足以充分表达

### 优化建议

- 尝试 Focal Loss 或 Weighted BCE 处理类别不平衡
- 延长训练 epochs（当前 50 可能不足）
- 考虑 SMOTE 或其他过采样策略

## 4. QM9DFT（量子化学属性预测）—— 🟢 优秀

**任务描述**：预测分子的量子化学 DFT 计算属性（如 HOMO/LUMO 能级、偶极矩、零点能等），是计算化学的基础任务。

### 评估指标详解

| 指标 | 数值 | 中文名 | 统计意义 |
|------|------|--------|---------|
| **Pearson r** | 0.9804 | 皮尔逊相关系数 | 极强的线性相关性，接近完美。r > 0.98 表明预测值与 DFT 计算值几乎呈完美线性关系。 |
| **Spearman r** | 0.9788 | 斯皮尔曼秩相关系数 | 单调关系同样极强，说明预测排序高度可靠。 |
| **R²** | 0.9605 | 决定系数 | 模型解释了 96% 的量子属性方差，属于极高水平。 |
| **MSE** | ~0.0000 | 均方误差 | 由于量子化学属性数值极小（~0.001 量级），误差几乎为零。这是量纲效应，不代表完美无误差。 |
| **MAE** | 0.0024 | 平均绝对误差 | 平均每个原子属性的预测误差仅 0.002 个 DFT 单位，在化学精度要求内（通常 1 kcal/mol ≈ 0.001 Hartree）。 |

### 结论

> QM9DFT 是四个任务中表现最优秀的。皮尔逊相关系数高达 0.98，R² = 0.96，MAE 仅 0.002。Uni-Mol 对量子化学属性的预测极为精准，这得益于 QM9 数据集质量高（DFT 计算精确）且样本量大。

## 总结与建议

### 任务表现排名

```
优秀程度排序（按核心指标）：
QM9DFT (r=0.98) > ESOL (r=0.95) > BBBP (AUC=0.90) > Tox21 (AUC=0.65)
```

### 行动建议

| 任务 | 状态 | 建议 |
|------|------|------|
| **QM9DFT** | ✅ 可用 | 直接用于下游计算化学任务，性能已达发表水平 |
| **ESOL** | ✅ 可用 | 效果优秀，可直接集成到药物溶解度预测流程 |
| **BBBP** | ⚠️ 可用但需优化 | 表现良好，MCC 偏低可进一步调优 |
| **Tox21** | ⚠️ 需优化 | 建议增大学习率、延长训练、使用 Focal Loss 处理类别不平衡 |

### 核心发现

1. **Uni-Mol 在量子化学属性预测（QM9DFT）和溶解度预测（ESOL）上表现优秀**，达到或接近领域先进水平
2. **BBBP 二分类任务表现良好**，AUC > 0.85，但在不平衡数据上 MCC 有提升空间
3. **Tox21 多标签分类任务偏弱**，准确率具有欺骗性，需关注 AUC/MCC 等不平衡数据专用指标
4. **类别不平衡是 Tox21 表现弱的根本原因**，建议采用加权损失函数或过采样策略
